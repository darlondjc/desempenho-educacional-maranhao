Objetivo Central

Objetivo Central

Apresentar os conceitos de Python mobilizados na pesquisa, exemplificando a implementação em classes e métodos que compõem a lógica do projeto.

Problema de Pesquisa

  • Tema: Padrões de Desempenho Educacional nos Municípios Maranhenses
  • Pergunta: Quais fatores estruturais e escolares se associam ao IDEB entre os municípios?
  • Recorte: Municípios maranhenses, com os seguintes datasets:
    • IDEB e Censo Escolar (2023)
    • IBGE e IDHM (2010)

Arquitetura em Python

  • Pipeline de dados: base → raw → trusted → reports
  • Módulos do projeto:
    • src/ingest.py — ingestão e engenharia de indicadores
    • src/transform.py — limpeza e integração das bases
    • src/analytics.py — clusterização, comparação e modelagem
    • src/util.py — utilitários de tempo e controle
  • Paradigma: pipeline funcional orientado a dados

Exemplos de Código

Exemplo 1: Divisão Segura

# Divisão por zero gera inf ou erro; pd.NA propaga o valor ausente de forma controlada ao longo do pipeline
def safe_divide(
    numerator: pd.Series,
    denominator: pd.Series,
    factor: float = 1.0,
) -> pd.Series:
    denominator = denominator.replace(0, pd.NA)   # substitui 0 por NA
    return (numerator / denominator) * factor

Evita divisão por zero sem interromper o pipeline — retorna NA onde o denominador é inválido.

Exemplo 2: Padronização do identificador do município

def normalize_municipio_code(series: pd.Series) -> pd.Series:
    return (
        pd.to_numeric(series, errors="coerce")  # converte, força NA em falhas
        .astype("Int64")                         # inteiro que aceita valores nulos
        .astype("string")                        # string para join/merge seguro
    )

Bases distintas (IDEB, Censo, IBGE) codificam CO_MUNICIPIO como int, float ou string

Sem padronização de tipo, joins entre bases públicas podem falhar de forma silenciosa.

Exemplo 3: Agregação Municipal com Pandas

indicadores = (
    censo.groupby("CO_MUNICIPIO", as_index=False)
    .agg(
        NO_MUNICIPIO              = ("NO_MUNICIPIO",   "first"),
        QT_ESCOLAS                = ("CO_ENTIDADE",    "nunique"),
        PCT_ESCOLAS_COM_INTERNET  = ("IN_INTERNET",    "mean"),
        QT_MAT_BAS                = ("QT_MAT_BAS",     "sum"),
        QT_TUR_BAS                = ("QT_TUR_BAS",     "sum"),
    )
    .assign( # multiplica cada item por 100 para ficar em escala percentual (0 a 100)
        PCT_ESCOLAS_COM_INTERNET  = lambda x: x["PCT_ESCOLAS_COM_INTERNET"] * 100,
        MEDIA_ALUNOS_POR_TURMA   = lambda x: x["QT_MAT_BAS"] / x["QT_TUR_BAS"],
    )
)

Groupby + agg aplica múltiplas funções em uma só: passagem vetorizada → sem laços explícitos por município

As regras de negócio ficam explícitas na estrutura do código → sem laços manuais.

Exemplo 4: Limpeza de Valores do IDEB

# o INEP publica valores como "ND", "(*)", "--" para
# municípios sem divulgação — converter sem tratar gera NaN
valores_nulos = ["ND", "ND*", "ND***", "(*)", "(**)", "--", "-", ""]

for col in cols_numericas:
    df_ideb[col] = df_ideb[col].replace(valores_nulos, np.nan)
    df_ideb[col] = df_ideb[col].astype(str).str.strip()
    df_ideb[col] = df_ideb[col].replace("nan", np.nan)
    df_ideb[col] = df_ideb[col].str.replace(",", ".", regex=False)
    df_ideb[col] = pd.to_numeric(df_ideb[col], errors="coerce")

Dados administrativos heterogêneos transformados em base analítica consistente e tipada.

Modelagem e Análise

Exemplo 5: Clusterização com KMeans

# O número ideal de clusters não é definido arbitrariamente
inertias, silhouettes = [], []
for k in range(2, 11):
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = km.fit_predict(X_scaled)         # instancia + ajusta + prediz
    inertias.append(km.inertia_)
    silhouettes.append(silhouette_score(X_scaled, labels))

best_k = range(2, 11)[silhouettes.index(max(silhouettes))]
print(f"Melhor k pelo Silhouette: {best_k}")

KMeans é um algoritmo de agrupamento, clusterização. O número de clusters é definido por critério quantitativo, com ajuda do Silhouette Score — não por escolha arbitrária.

Silhouette Score mede coesão interna e separação entre grupos para cada valor de k
